- Install -

1. Install Forge 1.8.9

2. Launch Forge once, then close Minecraft

3. Put aspieware.jar inside the mods folder located at "%AppData%\Roaming\.minecraft\mods"

4. Relaunch Minecraft

5. If the main menu has changed, installation is complete!

====================

- Configs -

Loading official configs is done via the command .config onlinelist and .config onlineload "configname"

====================
